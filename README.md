# Medicine Inventory Tracker

A full-stack Medicine Inventory Management System built using React, JavaScript, Node.js, Express, and MongoDB.

## Features

* Add new medicines
* Update medicine stock
* Delete medicines
* Search medicines quickly
* Track expiry dates
* Low-stock alerts
* Responsive user interface
* REST API integration

## Tech Stack

### Frontend

* React.js
* JavaScript
* HTML/CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Installation

### Clone the Repository

```bash
git clone https://github.com/mondalankan2007-a11y/Medicine-Inventory-Tracker.git
cd Medicine-Inventory-Tracker
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run start:backend
```

### Start Frontend

```bash
npm run start:frontend
```
