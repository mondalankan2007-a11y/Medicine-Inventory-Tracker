# Medicine Inventory Tracker

A full-stack Medicine Inventory Management System built using React, JavaScript, Node.js, Express, and MongoDB. The application helps pharmacies and healthcare providers efficiently manage medicine stock, monitor inventory levels, and keep track of available medicines.

---

## Features

* Add new medicines to inventory
* Update medicine details and stock quantity
* Delete medicines from inventory
* Search and filter medicines
* Real-time inventory management
* User-friendly dashboard
* REST API integration
* Responsive design for desktop and mobile devices

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/mondalankan2007-a11y/Medicine-Inventory-Tracker.git
cd Medicine-Inventory-Tracker
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file in the backend directory and add the required environment variables.

Example:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### 4. Start the Backend Server

```bash
npm run start:backend
```

### 5. Start the Frontend Application

```bash
npm run start:frontend
```

The application will be available at:

```text
http://localhost:3000
```
